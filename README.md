# 🛒 Amazon Clone

A fully responsive Amazon-inspired e-commerce website built using **HTML, CSS, and JavaScript**. This project recreates the core user interface of Amazon, including the homepage, navigation bar, product listings, shopping cart functionality, and responsive design.

---

## 📌 Features

* 🏠 Amazon-like Homepage
* 🔍 Search Bar UI
* 🛍️ Product Listing Section
* 🛒 Shopping Cart
* ❤️ Add to Cart Functionality
* 📱 Fully Responsive Design
* 🎨 Modern UI inspired by Amazon
* ⚡ Interactive JavaScript Features
* 📦 Product Cards with Images, Price & Details
* 🔥 Smooth User Experience

---

## 🛠️ Tech Stack

**Frontend**

* HTML5
* CSS3
* JavaScript (ES6)
